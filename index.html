AwesomeFramework by komagata

AwesomeFramework is PHP application framework of an extremely simple Plain Old PHP File base.

AwesomeFramework is PHP application framework of an extremely simple Plain Old PHP File base. * PHP4, PHP5 compatible! (maybe PHP3, PHP6 also...). AwesomeFramework never deserts PHP4. AwesomeFrameworkは昔ながらのPHPファイルベースのWebアプリケーションフレームワークです。 ※PHP4, PHP5*ちらでも動作します。（そしておそらくPHP3やPHP6でも･･･） AwesomeFrameworkは決してPHP4を見捨てません。
